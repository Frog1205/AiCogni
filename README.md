# AiCogni

AI 思维训练、AI 认知与工具链系统搭建的持续记录项目。

## 目录结构

```
AiCogni/
├── docs/          GitHub Pages 网站源码（当前发布：AI 工具链对比页）
├── notes/         AI 思维训练与认知笔记
├── toolchain/     工具链配置、脚本、评测记录
├── experiments/   阶段性实验记录
└── ai-toolchain-comparison-v3_1.html   工具链对比页原始版本（v3.1，归档）
```

## 在线预览

工具链对比页（国内线 / 海外工具链推荐）已通过 GitHub Pages 发布：

https://frog1205.github.io/AiCogni/

## 更新方式

网站源码位于 `docs/index.html`，修改后提交并推送到 `main` 分支，GitHub Pages 会自动重新发布。
